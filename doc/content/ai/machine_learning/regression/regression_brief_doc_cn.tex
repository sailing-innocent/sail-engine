\paragraph{回归问题}

假设我们拥有一个数据集包括N个自变量数据$x_i$和对应的因变量数据$y_i$，我们希望得到一个模型$f(x)$使得这个模型可以最好地贴近我们的数据集。这里就涉及三个核心概念：

\begin{enumerate}
    \item 数据集$\{x_i,y_i|N\}$
    \item 模型$y=f(x)$
    \item 接近度量$L(\theta)=L(y_i,f(x_i))$
\end{enumerate}
