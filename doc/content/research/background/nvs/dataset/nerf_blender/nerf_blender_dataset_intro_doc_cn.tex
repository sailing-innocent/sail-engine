\paragraph{NeRF Blender 数据集}

NeRF-Blender数据集是NeRF原论文中使用开源建模软件Blender合成的数据集，包括八个场景

\begin{enumerate}
    \item 椅子 chair 
    \item 鼓 drums
    \item 灌木 ficus 
    \item 热狗 hotdog 
    \item 乐高 lego
    \item 材质球 material
    \item 麦克风 mic
    \item 船模 ship 
\end{enumerate}
